# Gestion-d-une-Bibliotheque
Développement d’une application de gestion  d'une Bibliothèque
🔹 Architecture et fonctionnalités
L’application est organisée autour de deux espaces distincts :
👨‍💼 Espace Responsable (Administrateur)
Gestion du catalogue (ajout, recherche, suppression)
Validation et suivi des emprunts
Gestion des adhérents
Consultation de l’historique et des statistiques
👤 Espace Adhérent:
Recherche et consultation du catalogue
Demande et suivi des emprunts
Restitution des livres
Consultation de l’historique personnel
🛠️ Technologies et concepts utilisés:
Langage C 
Programmation modulaire
Structures de données
Gestion de fichiers
Interfaces console interactives
